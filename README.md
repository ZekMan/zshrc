## Installation ##

    cd /tmp
    git clone git://github.com/ZekMan/zshrc.git
    cd zshrc
    mv .zshrc .zsh/ ~/
    mkdir ~/.zsh/plugins
    cd ~/.zsh/plugins
    git clone git://github.com/zsh-users/zsh-syntax-highlighting.git
    sudo usermod USERNAME -s /usr/bin/zsh
